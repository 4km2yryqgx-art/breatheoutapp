# 🏋️ FitApp — фитнес Telegram Mini App

Полноценное фитнес-приложение внутри Telegram: анкета с ИИ-расчётом КБЖУ,
дневник тренировок, база упражнений, графики прогресса, геймификация (XP,
уровни, ачивки), Premium-подписка за Telegram Stars и промокоды.

---

## 📂 Структура проекта

```
fitness-tma/
├── app/
│   ├── main.py            # FastAPI-приложение, точка входа
│   ├── bot.py              # Telegram-бот (aiogram 3.x)
│   ├── config.py           # Настройки + мастер-промокоды
│   ├── database.py         # Модели SQLAlchemy
│   ├── schemas.py          # Pydantic-схемы запросов
│   ├── security.py         # Валидация Telegram initData
│   ├── ai_logic.py         # Расчёт КБЖУ, генерация тренировок, XP
│   ├── seed_data.py        # База упражнений (90+ шт.) и ачивок
│   └── routers/             # API-эндпоинты (профиль, тренировки, планы...)
├── frontend/
│   ├── index.html           # Разметка всех экранов Mini App
│   ├── css/style.css
│   └── js/                  # Логика фронтенда (без сборки, чистый JS)
├── requirements.txt
├── render.yaml               # Автодеплой на Render
└── .env.example
```

## ⚠️ Важно перед стартом

- Это **рабочий, но MVP-уровня** проект — база упражнений содержит ~90
  проверенных упражнений (не 300+), но легко расширяется: просто добавляй
  строки в `app/seed_data.py` в список `EXERCISES`, либо через кнопку
  "Добавить своё упражнение" в самом приложении.
- Для бесплатного и быстрого старта используется **SQLite** — она отлично
  подходит для теста, но у бесплатного плана Render диск не гарантированно
  постоянный между деплоями. Для реальных пользователей настоятельно
  рекомендую подключить **Supabase Postgres (бесплатный тариф)** — просто
  поменяй `DATABASE_URL` (см. шаг 5).

## 🚀 Шаг 1. Создай бота в @BotFather

1. Открой Telegram, найди **@BotFather**.
2. Отправь `/newbot`.
3. Введи имя бота (то, что будет отображаться) — например, `FitApp Coach`.
4. Введи технический username бота — обязательно заканчивается на `bot`,
   например `myfitapp_bot`.
5. BotFather пришлёт токен вида:
   ```
   123456789:AAExampleTokenAbCdEfGhIjKlMnOpQrStUvWx
   ```
   **Это и есть твой `BOT_TOKEN`** — сохрани его, он понадобится на шаге 4.

---

## 🌐 Шаг 2. Задеплой код бесплатно на Render

1. Зарегистрируйся на [render.com](https://render.com) (можно через GitHub).
2. Залей эту папку с проектом к себе в GitHub-репозиторий (создай новый
   репозиторий, загрузи туда все файлы).
3. В Render нажми **New → Web Service**, подключи свой GitHub-репозиторий.
4. Render автоматически подхватит `render.yaml`. Если не подхватит —
   настрой вручную:
   - **Runtime**: Python 3
   - **Build command**: `pip install -r requirements.txt`
   - **Start command**: `uvicorn app.main:app --host 0.0.0.0 --port $PORT`
   - **Plan**: Free
5. В разделе **Environment** добавь переменные (шаг 4 объясняет какие).
6. Нажми **Create Web Service** — через пару минут получишь публичный URL
   вида `https://fitness-tma.onrender.com`.

> Альтернативы Render: **Railway.app** (аналогичный процесс) или
> **Fly.io**. Логика та же: подключаешь репозиторий, задаёшь переменные
> окружения, деплоишь.

---

## 🔑 Шаг 3. Куда вставлять токен

В Render (или в файле `.env`, если запускаешь локально) задай переменные:

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | Токен из BotFather (шаг 1) |
| `WEBAPP_URL` | Публичный URL твоего сервера с Render (шаг 2), например `https://fitness-tma.onrender.com` |
| `DATABASE_URL` | `sqlite+aiosqlite:///./fitness.db` для быстрого старта |
| `PREMIUM_PRICE_STARS` | `100` (или своя цена в Stars) |

Локально — скопируй `.env.example` в `.env` и впиши свои значения:
```bash
cp .env.example .env
# затем открой .env и вставь токен
```

## 📱 Шаг 4. Настрой Telegram Mini App в BotFather

1. В BotFather отправь `/mybots`, выбери своего бота.
2. Зайди в **Bot Settings → Menu Button → Configure Menu Button**.
3. Введи URL — **тот же `WEBAPP_URL`**, который ты указал в переменных
   окружения (например `https://fitness-tma.onrender.com`).
4. Введи текст кнопки, например `Открыть FitApp`.

Теперь у пользователей рядом с полем ввода появится кнопка, открывающая
твоё приложение. Плюс сам бот при команде `/start` присылает отдельную
кнопку "🏋️ Открыть FitApp" (это уже реализовано в коде `app/bot.py`).

**Дополнительно** (не обязательно, но красиво): через `/mybots → Edit Bot →
Edit Description/About/Botpic` можно задать красивое описание и аватар
бота.

## ⭐ Шаг 5. Настройка оплаты Telegram Stars

Ничего дополнительно настраивать в BotFather не нужно — Stars работают
"из коробки" для всех ботов. В коде уже всё реализовано:

- В Mini App на экране **Premium** пользователь жмёт "Купить за 100 Stars".
- Открывается нативное окно оплаты Telegram.
- После успешной оплаты бот (`app/bot.py`, обработчик `successful_payment`)
  автоматически активирует Premium и присылает пользователю уникальный
  промокод, который можно подарить другу.

### Мастер-промокоды для друзей

Открой `app/config.py` и посмотри словарь `MASTER_PROMOCODES`:

```python
MASTER_PROMOCODES = {
    "FRIENDS2026": {"type": "lifetime", "max_uses": None},
    "TESTMONTH": {"type": "month", "max_uses": 50},
}
```

Добавляй сюда любые свои коды — просто отредактируй файл и передеплой
(Render подхватит изменения автоматически при пуше в GitHub).

## 🐘 (Рекомендуется) Переезд на Supabase Postgres — бесплатно и надёжно

1. Зарегистрируйся на [supabase.com](https://supabase.com), создай проект.
2. В **Project Settings → Database** найди строку подключения
   (Connection string, режим **Session pooler** или **Direct connection**).
3. Замени в ней `postgresql://` на `postgresql+asyncpg://`.
4. Впиши получившуюся строку в переменную `DATABASE_URL` на Render.
5. Передеплой сервис — таблицы создадутся автоматически при первом запуске.

## 🧪 Локальный запуск (для разработки)

```bash
pip install -r requirements.txt
cp .env.example .env   # впиши свой BOT_TOKEN
uvicorn app.main:app --reload
```

Открой `http://localhost:8000` в браузере — увидишь заглушку "открой через
Telegram" (это нормально: Mini App работает по-настоящему только внутри
Telegram, так как проверяется подпись initData). Чтобы протестировать
именно внутри Telegram при локальной разработке — используй
[ngrok](https://ngrok.com) или аналог, чтобы получить публичный HTTPS-URL,
и укажи его как `WEBAPP_URL` и в BotFather.

## 📲 Как пользователю вывести FitApp на главный экран iPhone/Android

1. Открой бота в Telegram и нажми кнопку **"Открыть FitApp"**.
2. В открывшемся Mini App нажми на **три точки (⋮)** в правом верхнем углу
   (на Android) или на **название приложения вверху экрана** (на iPhone).
3. Выбери пункт **"Добавить на главный экран" / "Add to Home Screen"**.
4. Подтверди — иконка FitApp появится на рабочем столе телефона, как
   обычное приложение, и будет открывать Mini App в один тап.
